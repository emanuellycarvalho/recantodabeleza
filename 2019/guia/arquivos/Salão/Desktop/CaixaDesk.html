Diretrizes de Caixas de Mensagem

  • Evite usar terminologia técnica ou frases complexas;
  • Apresente uma caixa de mensagem para cada condição específica;
  • A mensagem deve ser clara, concisa e utilizar termos do vocabulário do usuário;
  • Não confie em mensagens default do sistema de mensagens, tais como Kernel INT 24;
  • Seja breve, porém completo, limite a mensagem a duas ou três linhas (mais ou menos 60 caracteres);
  • Quando maiores explicações forem necessárias e a tela já contém muitos dados, utilize um botão para mostrar mais detalhes;
  • Se houver uma sequência de eventos, a ordem das palavras na frase deve corresponder a sequência temporal das ocorrências.
