Categoria: Caixas de Mensagem

Orientação: Mensagem de Advertência

  • Utilize Mensagens de Advertência para prevenir o usuário de uma ação potencialmente perigosa, irreversível, instável ou destrutiva;
  • Mostre informações sobre o que está acontecendo:
    • Observe que mostramos dados acerca do cliente a ser excluído que o diferenciam dos demais: Nome, E-mail, CPF e Telefone.
  • Apresente a mensagem como uma pergunta objetiva;
  • Coloque cada botão de ação da caixa como uma resposta diferente à pergunta;
  • Defina o botão que representa a ação menos perigosa como padrão para Enter.
    • Neste projeto exemplo, Enter seleciona resposta Não.
