Orientação: Visualizar

  • Lembre de seguir as diretrizes de Formulário ao construir uma tela de Visualização;
  • Crie a tela de Visualização para ser semelhante à tela de Novo, assim o usuário sentirá familiaridade com a tela;
  • As orientações acerca do Formulário de Novo aplicam-se ao Formulário de Visualizar:
    • Com a diferença que os campos devem vir já preenchidos;
    • Além disso, estes campos devem estar desabilitados, para o usuário não se confundir.
