Categoria: Formulários

Orientação: Campo de Texto

  • Também pode ser chamado de TextField.
  • Utilize o Campo de Texto quando:
    • Quiser permitir ao usuário introduzir e manipular caracteres.
  • Utilize-o se o tamanho da opção não exceder 40 caracteres;
  • É possível apresentar rótulos descritivos para dados de formato particular no rótulo:
    • Temos o exemplo do campo data: dd/mm/aaaa; além do exemplo de e-mail do cliente.
