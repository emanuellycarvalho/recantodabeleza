Categoria: Formulários

Orientação: Rótulo

  • Também pode ser chamado de Label.
  • Utilize o Rótulo quando:
    • Houver um elemento a ser descrito, como um campo de texto.
  • Os rótulos devem indicar quando um campo é obrigatório:
    • Neste projeto exemplo, utilizamos um * para isso;
    • É possível utilizar a palavra obrigatório também.
  • Evite o uso de hífen, porque isso reduz a velocidade de leitura;
  • Escolha nomes óbvios e claros, de forma a serem compreendidos rapidamente;
  • Há duas formas de alinhar os rótulos aos seus elementos correspondentes:
    • Posicione-os à esquerda dos elementos se não houver muitas opções;
    • Coloque-os acima dos elementos se o formulário tiver largura muito grande.
  • Utilize caixa alta para a primeira letra de cada palavra e caixa baixa para as demais.
    • Por exemplo, escreva Nome Completo em vez de Nome completo.
