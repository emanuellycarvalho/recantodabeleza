Categoria: Diretrizes Gerais de Layout

Orientação: Ícones

  • Função: representar objetos ou tarefas;
  • Mantenha padrões de ícone por todo o sistema;
  • Recomenda-se o uso de metáforas do mundo real;
  • Classifique os ícones por estilos ou aparência;
  • Procure não adicionar elementos decorativos sem necessidade;
  • Evite os contornos em preto porque aumentam a poluição visual;
  • Use objetos grandes, linhas grossas e áreas simples para distinguir os ícones;
  • Evite que as partes mais significativas do ícone sejam pequenos em comparação com o total;
  • Recomenda-se que o ícone tenha uma aparência simplificada:
    • Um ícone composto de muitas partes pode confundir o usuário;
  • Utilize um estilo comum para todos os ícones;
    • Reconhecimento e memória do usuário são importantes são no projeto de ícones;
  • Inclua cores com discrição:
  • É suficiente o uso de, no máximo, cinco cores (incluindo o preto, o branco e/ou o cinza);
  • Recomenda-se que seja projetado um grid para organizar todos os elementos que compõem o ícone:
    • Isso é importante para estabelecer padrões de linhas horizontais, verticais e oblíquas;
    • Além de um número limitado de tamanhos para os objetos que compõem aquele ícone;
  • Considere o potencial cultural do impacto dos ícones:
    • O que pode ter um certo significado em uma país ou cultura pode ter significados imprevistos em outro;
    • É melhor evitar letras ou palavras, se possível, porque tornam os desenhos difíceis de aplicar em outras culturas;
  • Onde for possível e apropriado, use perspectiva e dimensão (luz e sombra) para melhor comunicar a representação do mundo real.
