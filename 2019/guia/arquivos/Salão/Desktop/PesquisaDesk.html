Diretrizes de Pesquisa

  • Tela de Pesquisa: esta tela tem a função de procurar um item com mais precisão, caso não seja encontrado na busca pela própria caixa de seleção:
    • Por exemplo, se houver dois ou mais clientes com o mesmo nome, pode-se tirar essa dúvida em uma janela que mostre dados com menos chance de ambiguidade, como Telefone e E-mail;
    • Procure utilizar a mesma tela sempre que for pesquisar a mesma categoria de objeto;
    • Siga as mesmas orientações de Filtro e Paginação de CRUD;
    • Apenas habilite o botão Escolher enquanto um item estiver selecionado e desabilite-o quando um novo filtro for adicionado ou removido, porque esta tela tem um fluxo simples: somente selecionar o cliente e fechar a tela logo depois;
    • Permita ao usuário sair desta tela com um botão específico além do Fechar do sistema operacional, neste projeto exemplo, ele se chama Voltar.
  • Campos de Pesquisa: estes são os campos de texto e botões que permitirão adicionar um cliente àquela tela, algumas orientações:
    • Mantenha um padrão destes campos em todas as telas;
    • Torne a caixa de seleção pesquisável a fim de facilitar a busca;
    • Insira um botão de Pesquisar para abrir uma janela própria para pesquisa;
    • Coloque um botão de Novo para fazer um cadastro, caso não seja encontrado o item procurado.
