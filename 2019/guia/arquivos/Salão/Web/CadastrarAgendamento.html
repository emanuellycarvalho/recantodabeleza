Categoria: Calendário

Orientação: Agendar Agendamento

  • Lembre-se de observar as orientações de Formulario ao construir uma tela de Agendar Agendamento;
  • Observe que há uma separação ( Agrupamento) entre os dados do agendamento: uma parte que armazena dados do dia, horário e cliente e outra que armazena dados dos serviços prestados.
  • O cliente do salão pode querer receber mais de um serviço, como um corte de cabelo e manicure, por exemplo. Por esse fato, o sistema deve permitir que vários serviços sejam cadastrados e, eventualmente, excluídos.
