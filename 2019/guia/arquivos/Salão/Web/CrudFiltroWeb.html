Orientação: Filtrar

  • Lembre-se de observar as orientações de Caixas de Diálogo ao construir uma tela de Filtrar;
  • Esta função permite reduzir os itens exibidos na lista ao filtrar, por campos previamente estabelecidos, todos os registros armazenados, diferenciando-se da funcionalidade de Pesquisar, que filtra apenas os registros que estão carregados na página que o usuário se encontra. Algumas orientações:
    • Ao filtrar, exiba o botão Limpar para permitir retornar para a exibição de todos os itens cadastrados;
    • E também esconda o botão Limpar depois de ele ser acionado.
    • A partir do preenchimento dos campos de filtro e acionamento do botão de Aplicar, o filtro fará a união ("ou") dos filtros preenchidos e o sistema carregará uma nova tabela que possui os registros dos filtros.
      • Atenção: no projeto exemplo, não se usa Banco de Dados. Com isso, o Filtrar só funciona se nos campos de filtro estivem preenchidos textos iguais aos armazenados. Ao se usar Banco de Dados, o Filtrar deve permitir que se filtre por partes dos textos armazenados. Por exemplo: se no campo Nome estiver preenchido "Maria", o retorno do Filtrar deve conter nomes como "Ana Maria" e "Maria Paula". Para isso, use, na SQL, like '%Maria%'.
