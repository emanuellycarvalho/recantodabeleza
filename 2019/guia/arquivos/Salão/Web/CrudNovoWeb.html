Categoria: CRUD

Orientação: Novo

  • Lembre de seguir as diretrizes de Formulário ao construir uma tela de Cadastro;
  • Observe que é possível cadastrar uma nova cidade enquanto se preenche o Cadastro de Cliente:
      O botão de "+" deve ficar ao lado do select de cidade, a fim de parecer intuitivo ao usuário;
    • Ao se clicar no "+", abrirá uma Caixa de Diálogo do tipo modal, que cadastrará a nova cidade, fazendo com que o usuário não precise sair da tela de Cadastro de Cliente para ir ao Cadastro de Cidade, perdendo os dados já preenchidos;
