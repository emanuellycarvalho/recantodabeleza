Categoria: Formulários

Orientação: Botão

  • Mantenha padrões de botão por todo o sistema:
    • Por exemplo: as cores azul e verde foram usadas para ações negativas e positivas, respectivamente;
    • Em outros casos, foram utilizados botões com fundo de outra cor, como o preto para o fechar, e, ainda, outros botões com ícones intuitivos.
  • Escolha uma cor de fundo diferente para os botões;
  • Lembre-se: as diretrizes de ícones e cores aplicam-se também aos botões.
  • Dimencione-os num formato quadrangular ou retangular:
    • Para assinalar os limites da área interativa.
    • É possível utilizar cantos arredondados para um visual mais moderno.
  • Defina o tamanho do botão para ser facilmente interpretado no contexto da página:
    • Não muito grande, porque chamaria muita atenção;
    • Nem muito pequeno, uma vez que isso dificulta o usuário de acessá-lo.
  • Coloque os botões à direita ou ao centro:
    • Em formulários, eles devem ficar após os campos a que se referem.
      Exemplo: o botão de adicionar uma nova cidade fica logo após o combobox de cidades já inseridas.
  • Permita que os botões seja acionados por meio de atalhos:
    • Atalhos com combinações como Ctrl ou Shift.
