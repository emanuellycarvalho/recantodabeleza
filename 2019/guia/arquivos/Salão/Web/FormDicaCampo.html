Categoria: Formulários

Orientação: Campo de Texto

  • Também pode ser chamado de input.
  • O tipo do Campo de Texto deve levar em consideração a informação que se deseja inserir nele:
    • Tipo text(texto) para nomes;
    • Tipo date(data) para datas;
    • Tipo e-mail para e-mail;
    • Tipo password(senha) para senha.
    • Isso contribui para prevenir erros do usuário.
  • Utilize o Campo de Texto quando:
    • Quiser permitir ao usuário introduzir e manipular caracteres.
  • Utilize-o se o tamanho da opção não exceder 40 caracteres;
  • É possível apresentar rótulos descritivos para dados de formato particular no rótulo:
    • Tem-se o exemplo do campo data: dd/mm/aaaa; além do exemplo do e-mail do cliente.
