Categoria: Formulários

Orientação: Caixa de Seleção

  • Também pode ser chamado de ComboBox ou Select.
  • Utilize a Caixa de Seleção quando:
    • Deve ser escolhida apenas uma opção;
    • Quiser que as opções ocupem pouco espaço na tela;
    • O número opções for grande, ou seja, maior do que 7 (sete).
  • É importante que a Caixa de Seleção tenha um filtro interno, a fim de facilitar a procura por um item quando houver uma longa lista de itens.
  • Defina uma ordem lógica para as opções, a qual pode ser:
    • Frequência de uso;
    • Ordem cronológica;
    • Letras do alfabeto;
    • Tamanho do texto das opções:
      • Da menor para a maior;
      • Ou da maior para a menor.
