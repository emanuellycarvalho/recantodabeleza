Diretrizes de Login


  • Recuperar Senha: lembre-se de permitir ao usuário de um Sistema Web poder recuperar a senha, caso ele venha a esquecer.
  • Título: procure sempre identificar para o usuário qual tela esta sendo utilizada.
  • Formulário: a tela de login contém campos de preenchimento, consequentemente, as diretrizes referentes a formulários também se aplicam aqui.
    • Por exemplo, as orientações de informar ao usuário sobre preenchimento dos campos e incluir rótulos.
  • Mensagem Genérica: em caso de login incorreto, não avise qual das informações fornecidas está errada. Em vez disso, diga que qualquer uma delas pode estar errada, para mais segurança.
    • Veja como "Usuário ou senha incorretos." não diz nem se é o e-mail do usuário e nem se é a senha.
  • Botão Cancelar: lembre-se de incluir sempre uma forma de sair de uma tela, mesmo que exista o botão padrão de fechar tela do sistema operacional.
    • Em nosso padrão de cores de botão, o Cancelar tem cor azul que se optou como sendo a cor que indica uma ação negativamente realizada.
