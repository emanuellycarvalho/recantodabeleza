Diretrizes de Mestre-Detalhe em projeto para desktop

  • Informação Mestre: este é o objeto da ação que se relaciona a todos os itens listados na tela. Algumas orientações:
    • Separe os campos-mestre em um grupo;
    • Coloque-os no início da tela, para ser facimente identificada a hierarquia das informações.
  • Informações Relacionadas: estes são os itens que têm de estar pertencentes ao mestre. Algumas orientações:
    • Separe cada categoria de itens num grupo;
    • Liste as informações por meio de uma tabela.
  • Botões de Tabela: coloque os botões de Adicionar e de Remover itens perto da tabela:
    • Como os botões não têm rótulo, mantenha uma Dica, também chamada de Bolha de Ajuda, para o usuário saber qual é a função deles.
