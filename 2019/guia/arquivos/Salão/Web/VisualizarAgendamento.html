Categoria: Calendário

Orientação: Visualizar Agendamento

  • Lembre-se de observar as orientações de Caixas de Diálogo ao construir uma tela de Visualizar Agendamento;
  • A Caixa de Diálogo de Visualizar Agendamento deve conter os dados do agendamento de determinado cliente, como o Nome do Cliente, a data do atendimento, bem como o horário inicial e o horário final
  • Atenção: a Caixa de Diálogo de Visualizar Agendamento deve permitir que se edite e exclua um agendamento. No caso do projeto exemplo, contudo, isso não ocorre.
